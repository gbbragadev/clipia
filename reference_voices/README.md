# Reference Voices

Coloque aqui um arquivo .wav de 6-15 segundos com voz pt-BR para clonagem XTTS.

Requisitos:
- Formato: WAV, 22050 Hz ou 24000 Hz
- Duracao: 6 a 15 segundos
- Audio limpo, sem ruido de fundo
- Voz clara e natural em portugues brasileiro

Renomeie o arquivo para `narrator_ptbr.wav`.
